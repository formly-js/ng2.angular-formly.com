# ng2.angular-formly.com
Website for [Angular Formly 2](https://ng2.angular-formly.com)

# Development
```
> npm install -g harp
```

```
> harp server # Will host a local dev server
```

# Production

```
> harp compile # Will output a www/ folder
> firebase deploy # Will go live
```
